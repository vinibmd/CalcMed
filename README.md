CALCULADORA DE DOSES PARA BOMBA DE INFUSÃO CONTÍNUA

Web app para conversão de taxa de infusão em mL/h para mcg/kg/min a partir da concentração da solução e do peso do paciente.

FORMA DE UTILIZAÇÂO:
1. Selecione a medicação.
2. Preencha os campos com as informações solicitadas.
3. Clique em "Calcular"
4. Para limpar os campos, clique em "Resetar".

A dose infundida será mostrada abaixo do botão "Calcular" juntamente com a dose alvo (habitual) para determinada medicação.
